## 📂 프로젝트 구조 안내

본 저장소는 크게 실제 구동되는 **[소스 코드 및 매니페스트 폴더]**들과, 아키텍처 및 의사결정을 서술한 **[기술 문서 폴더(`docs/`)]**로 나뉘어 있습니다.

```text
ezcaretech-mlops-task/
├── .github/                  # [CI/CD] GitHub Actions 빌드 및 테스트 파이프라인
├── api/                      # [로직] FastAPI 기반 모델 서빙 API 코드 (predict, health)
├── deployments/              # [인프라] Kubernetes 배포 매니페스트 (Deployment, Service, HPA)
├── monitoring/               # [모니터링] Prometheus & Grafana 도커 스택 설정
├── tests/                    # [검증] Pytest 및 부하 테스트 스크립트
├── Dockerfile                # API 서버 최적화용 Multi-stage 빌드 이미지
└── docs/                     # [★ 핵심 제출 산출물] 과제 요구사항 문서 (아래 상세설명 참고)
```
